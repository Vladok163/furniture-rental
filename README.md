# Аренда мебели — веб-приложение

React + Vite + Supabase + Vercel

## Быстрый старт

### 1. Supabase

1. Зайди на [supabase.com](https://supabase.com) → создай новый проект
2. Открой **SQL Editor** → вставь содержимое файла `schema.sql` → нажми Run
3. Скопируй из **Project Settings → API**:
   - `Project URL`
   - `anon public` key

### 2. Локальный запуск

```bash
# Установи зависимости
npm install

# Создай файл .env (скопируй из .env.example)
cp .env.example .env
# Вставь свои значения из Supabase

# Запусти
npm run dev
```

Открой http://localhost:5173

### 3. Деплой на Vercel

1. Залей проект на GitHub
2. Зайди на [vercel.com](https://vercel.com) → Import Git Repository
3. В настройках проекта → **Environment Variables** добавь:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
4. Deploy → готово, получишь публичный URL

## Что умеет приложение

- **Дашборд** — активные заказы, свободная мебель, блок «сегодня возвращают», выручка за месяц
- **Заказы** — создание/редактирование, предупреждение о нехватке мебели на даты, предоплата и остаток
- **Клиенты** — поиск, история аренды, метка «Постоянный» при 2+ заказах, источник (Авито/Сарафан/Повтор)
- **Календарь** — цвет насыщенности по загруженности, клик на день показывает заказы
- **Склад** — свободно/занято прямо сейчас + таблица кто что взял
